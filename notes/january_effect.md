# January Effects

Here are the S\&P 500 returns for the month of January over the past eight years:

| Year | January Return |
| ---- | -------------- |
| 2017 | +2.58%         |
| 2018 | -2.89%         |
| 2019 | +5.83%         |
| 2020 | +0.12%         |
| 2021 | +2.49%         |
| 2022 | -2.90%         |
| 2023 | +3.15%         |
| 2024 | +4.20%         |

These figures represent the percentage change in the S\&P 500 index during each January from 2017 through 2024.
